# StudyFlow

StudyFlow 是一个资料驱动的 AI 学习闭环系统。它的首版目标不是做完整学习操作系统，而是把用户已有资料转成一条可执行、可验证、可复习的短学习链路。

当前 MVP 聚焦五步：

1. 导入资料
2. 自动切分学习单元
3. 基于原文讲解与问答
4. 完成轻量小测
5. 生成复习任务
